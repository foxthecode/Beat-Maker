## SECTION 1 — MOTEUR AUDIO : corrections `_mkRv()`, `updateReverb()`, init delay

### 1a. Remplacer `_mkRv()` dans la classe `Eng`

Remplace la méthode existante par cette version. Les changements : normalisation du gain de sortie par `(1 / nCombs)` au lieu du facteur fixe `0.015`, et ajout d'un stage d'early reflections (3 délais courts après les allpass) pour un son plus naturel et moins "en boîte".
```typescript
_mkRv(decay = 2, size = 0.5, type = 'room') {
  const sr = this.ctx.sampleRate;
  const scale = sr / 44100;
  const cfgs = {
    plate: { roomSize: 0.52,              damp: 0.82, preDelay: 0.002 },
    room:  { roomSize: 0.50 + size * 0.3, damp: 0.5 + size * 0.3, preDelay: 0.01 + size * 0.02 },
    hall:  { roomSize: 0.75 + size * 0.2, damp: 0.3 + size * 0.2, preDelay: 0.025 + size * 0.04 },
  };
  const cfg = cfgs[type] || cfgs.room;
  const { roomSize, damp } = cfg;
  const preDelaySamples = Math.floor(cfg.preDelay * sr);
  // Early reflections : 3 délais courts (7ms, 13ms, 23ms) mixés avant le Schroeder
  const erDelaysSamples = [Math.floor(0.007 * sr), Math.floor(0.013 * sr), Math.floor(0.023 * sr)];
  const totalSamples = Math.ceil(sr * (Math.min(8, decay) + cfg.preDelay + 0.1));
  const buf = this.ctx.createBuffer(2, totalSamples, sr);
  const combDelaysL = [1116, 1188, 1277, 1356, 1422, 1491, 1557, 1617].map(d => Math.floor(d * scale));
  const combDelaysR = combDelaysL.map(d => d + 23);
  const apDelays    = [556, 441, 341, 225].map(d => Math.floor(d * scale));
  const nCombs = combDelaysL.length;

  for (let ch = 0; ch < 2; ch++) {
    const data = buf.getChannelData(ch);
    const combDelays = ch === 0 ? combDelaysL : combDelaysR;
    const combBufs   = combDelays.map(d => new Float32Array(d));
    const combPos    = new Int32Array(combDelays.length);
    const combFilt   = new Float32Array(combDelays.length);
    const apBufs     = apDelays.map(d => new Float32Array(d));
    const apPos      = new Int32Array(apDelays.length);
    // Buffers early reflections
    const erBufs = erDelaysSamples.map(d => new Float32Array(d));
    const erPos  = new Int32Array(erDelaysSamples.length);

    for (let i = 0; i < totalSamples; i++) {
      const input = (i === preDelaySamples) ? 1.0 : 0.0;

      // Early reflections : sortie instantanée des 3 lignes de délai courtes
      let erOut = 0;
      const erGains = [0.7, 0.5, 0.4]; // amplitude décroissante
      for (let e = 0; e < erBufs.length; e++) {
        erOut += erBufs[e][erPos[e]] * erGains[e];
        erBufs[e][erPos[e]] = input;
        erPos[e] = (erPos[e] + 1) % erBufs[e].length;
      }

      // Schroeder : filtres en peigne
      let combOut = 0;
      for (let c = 0; c < nCombs; c++) {
        const b = combBufs[c], pos = combPos[c];
        const delayed = b[pos];
        combFilt[c] = delayed * (1 - damp) + combFilt[c] * damp;
        b[pos] = input + combFilt[c] * roomSize;
        combPos[c] = (pos + 1) % b.length;
        combOut += delayed;
      }
      // Normalisation correcte : facteur (1/N) au lieu de 0.015 fixe
      combOut *= (1 / nCombs);

      // Allpass
      let apOut = combOut;
      for (let a = 0; a < apBufs.length; a++) {
        const b = apBufs[a], pos = apPos[a];
        const delayed = b[pos];
        const w = apOut + delayed * 0.5;
        b[pos] = w;
        apPos[a] = (pos + 1) % b.length;
        apOut = delayed - 0.5 * w;
      }

      // Mix early reflections (50%) + queue Schroeder (50%)
      data[i] = apOut * 0.5 + erOut * 0.5;
    }
  }
  this.rv = buf;
}
```

### 1b. Remplacer `updateReverb()` pour éviter le freeze UI Android
```typescript
updateReverb(decay: number, size: number, type = 'room') {
  // Calcul différé : ne bloque pas le thread principal
  clearTimeout((this as any)._rvPending);
  (this as any)._rvPending = setTimeout(() => {
    this._mkRv(decay, size, type);
    if (this.gRvConv) {
      try { this.gRvConv.buffer = this.rv; } catch (e) { /* ctx suspendu */ }
    }
  }, 0);
}
```

## SECTION 3 — PERFORM FX (Live Pads) : corriger et compléter

Dans la fonction IIFE du PERFORM FX (`// ── Perform FX v10 ──`), effectue les corrections suivantes.

### 3a. Corriger `startRvHold` et `stopRvHold`

Remplace les deux fonctions existantes par :
```typescript
const startRvHold = () => {
  engine.init();
  if (perfRvHold.current) return;
  perfRvHold.current = true;
  const f = R.fx[target] || { ...DEFAULT_FX };
  // Sauvegarde des valeurs courantes
  perfRvPrevRef.current = { mix: f.rMix ?? 0, decay: f.rDecay ?? 1.5, size: (f as any).rSize ?? 0.5 };
  // Boost temporaire : +45 points de mix, decay légèrement allongé
  const boostMix = Math.min(100, (f.rMix ?? 0) + 45);
  if (!isMaster && target && engine.ch[target]?.rvSend) {
    const t2 = engine.ctx?.currentTime ?? 0;
    engine.ch[target].rvSend.gain.setTargetAtTime(boostMix / 100, t2, 0.02);
  }
  // Active la reverb globale si elle ne l'est pas
  engine.updateReverb(f.rDecay ?? 1.5, (f as any).rSize ?? 0.5, (f as any).rType || 'room');
};

const stopRvHold = () => {
  if (!perfRvHold.current) return;
  perfRvHold.current = false;
  const prev = perfRvPrevRef.current;
  if (prev && target && engine.ch[target]?.rvSend && engine.ctx) {
    const t2 = engine.ctx.currentTime;
    // Retour progressif au niveau original (200ms)
    engine.ch[target].rvSend.gain.setTargetAtTime(prev.mix / 100, t2, 0.08);
  }
  perfRvPrevRef.current = null;
};
```

### 3b. Corriger `startDlHold` et `stopDlHold`

const startDlHold = () => {
  engine.init();
  if (perfDlHold.current) return;
  perfDlHold.current = true;
  const f = R.fx[target] || { ...DEFAULT_FX };
  perfDlPrevRef.current = { mix: f.dMix ?? 0, time: f.dTime ?? 0.25, fb: f.dFdbk ?? 35 };
