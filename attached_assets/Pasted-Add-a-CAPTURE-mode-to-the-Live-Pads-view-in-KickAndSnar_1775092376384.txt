Add a CAPTURE mode to the Live Pads view in KickAndSnare.tsx.

════════════════════════════════════════════════════════
FEATURE OVERVIEW
════════════════════════════════════════════════════════
In the Live Pads view, the user plays pads freely while the metronome
runs. When they click CAPTURE, all hits recorded in the looper since
the last beat 1 (one full loop length) are quantized intelligently into
the current sequencer pattern, which then plays back immediately.

The quantization is "smart": for each hit, the algorithm finds the nearest
subdivision among [1/4, 1/8, 1/16, 1/32] and picks the one that minimizes
the distance — no fixed grid imposed.

--- STEP 3: Smart quantize + convert function ---

Add this function near toggleLoopRec and clearLooper:

  const captureToSequencer = () => {
    const L = loopRef.current;
    if (!L.events || L.events.length === 0) return;

    const loopDurMs = L.lengthMs > 0
      ? L.lengthMs
      : loopBars * sig.beats * (60000 / Math.max(30, bpm));

    // Available subdivisions in ms
    const beatMs = 60000 / Math.max(30, bpm);
    const subdivMs = [
      beatMs,           // 1/4
      beatMs / 2,       // 1/8
      beatMs / 4,       // 1/16
      beatMs / 8,       // 1/32
    ];

    // Number of 1/16 steps in the loop (base grid)
    const totalSteps = loopBars * sig.beats * 4; // 1/16 steps
    const stepMs = loopDurMs / totalSteps;

    // Build new pattern: for each track, an array of 0s
    const newPat = Object.fromEntries(
      ALL_TRACKS.map(t => [t.id, Array(totalSteps).fill(0)])
    );
    const newVel = Object.fromEntries(
      ALL_TRACKS.map(t => [t.id, Array(totalSteps).fill(100)])
    );

    L.events.forEach(ev => {
      if (!newPat[ev.tid]) return;

      // Find nearest subdivision for this hit
      let bestSnapMs = stepMs;
      let bestDist = Infinity;
      subdivMs.forEach(sub => {
        const snapped = Math.round(ev.tOff / sub) * sub;
        const dist = Math.abs(ev.tOff - snapped);
        if (dist < bestDist) {
          bestDist = dist;
          bestSnapMs = sub;
        }
      });

      // Convert snapped time to 1/16 step index
      const snappedMs = Math.round(ev.tOff / bestSnapMs) * bestSnapMs;
      const stepIdx = Math.round(snappedMs / stepMs) % totalSteps;

      // Write hit — keep highest velocity if multiple hits on same step
      newPat[ev.tid][stepIdx] = 1;
      const velPct = Math.round((ev.vel || 1) * 100);
      newVel[ev.tid][stepIdx] = Math.max(
        newVel[ev.tid][stepIdx],
        velPct
      );
    });

    // Push history for undo support
    pushHistory();

    // Resize sequencer to match loop length if needed
    const targetSig = TIME_SIGS.find(s => s.steps === totalSteps)
      ?? TIME_SIGS.find(s => s.label === '4/4')
      ?? TIME_SIGS[0];
    if (targetSig.steps !== sig.steps) chSig(targetSig);

    // Write into current pattern slot
    setPBank(pb => {
      const n = [...pb];
      const cp = { ...n[cPat] };
      ALL_TRACKS.forEach(t => {
        if (newPat[t.id].some(v => v > 0)) {
          cp[t.id] = [...newPat[t.id]];
        }
      });
      n[cPat] = cp;
      return n;
    });

    // Write velocities
    setStVel(sv => {
      const n = { ...sv };
      ALL_TRACKS.forEach(t => {
        if (newPat[t.id].some(v => v > 0)) {
          n[t.id] = [...newVel[t.id]];
        }
      });
      return n;
    });

    // Activate all captured tracks
    setAct(prev => {
      const next = [...prev];
      ALL_TRACKS.forEach(t => {
        if (newPat[t.id].some(v => v > 0) && !next.includes(t.id)) {
          next.push(t.id);
        }
      });
      return next;
    });

    // Stop looper, switch to sequencer, start playback
    stopLooper();
    setCaptureReady(false);
    switchView('sequencer');
    // Small delay to let React batch the state updates before starting
    setTimeout(() => {
      R.step = -1;
      nxtRef.current = engine.ctx.currentTime + 0.05;
      schLoop();
      setPlaying(true);
    }, 80);
  };

BEHAVIOR SUMMARY
════════════════════════════════════════════════════════
1. User opens Live Pads view, starts metro, plays pads freely
2. After one full loop bar completes, CAPTURE button appears and pulses
3. User clicks CAPTURE:
   - Each hit is snapped to the nearest subdivision (1/4, 1/8, 1/16, 1/32)
     using minimum-distance detection — no fixed grid imposed
   - Hits are written into the current pattern slot with their velocities
   - Tracks that received hits are automatically activated
   - Looper stops, view switches to Sequencer, playback starts immediately
   - Undo (Ctrl+Z) restores the previous pattern
