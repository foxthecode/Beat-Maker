## 🎯 CONCEPT

L'effet **SPEED** modifie le `playbackRate` de chaque `AudioBufferSourceNode` au moment du `play()`.

- **Speed = 1.0** → Normal (son identique)
- **Speed = 0.5** → Half-time (1 octave en dessous, durée doublée)
- **Speed = 2.0** → Double-time (1 octave au dessus, durée moitié)
- **Speed = 0.25** → Ultra-slow (2 octaves en dessous, son étiré)

C'est le comportement classique des samplers hardware : pitch et durée sont liés.

### Scope
- **MASTER** : un seul multiplicateur global qui affecte toutes les pistes
- **PER-TRACK** : chaque piste peut avoir son propre multiplicateur
- **UI** : choix du mode (Master / per-track) dans l'interface
- **Glide** : option toggle — quand activée, la transition entre deux valeurs est progressive (~150ms) au lieu d'instantanée

### 2. Ajouter les méthodes publiques (après `setBpm()` vers ligne 507)

```javascript
// ═══ SPEED methods ═══
/** Set master speed multiplier (0.25 → 4.0). 1.0 = normal. */
setSpeedMaster(rate) {
  this._speedMaster = Math.max(0.25, Math.min(4, rate));
}
/** Set per-track speed multiplier. null = use master. */
setSpeedTrack(id, rate) {
  if (rate === null || rate === undefined) {
    delete this._speedTrack[id];
  } else {
    this._speedTrack[id] = Math.max(0.25, Math.min(4, rate));
  }
}
/** Enable/disable glide between speed changes. */
setSpeedGlide(on) { this._speedGlide = !!on; }
/** Get effective speed for a track (per-track overrides master). */
_getSpeed(id) {
  return this._speedTrack[id] ?? this._speedMaster;
}
```

### 3. Slider snapping — comportement du `<input range>`

Le slider HTML standard n'a pas de "snap to value". Pour obtenir le snap sur les valeurs repères quand le doigt/souris est proche, ajouter cette logique dans le `onChange` :

Remplacer le `onChange` du slider par :

```javascript
onChange={e=>{
  let raw=parseInt(e.target.value)/100;
  // Snap to nearest marker if within ±0.04
  const snaps=[0.25,0.5,0.75,1,1.25,1.5,1.75,2,2.25,2.5,2.75,3];
  const closest=snaps.reduce((a,b)=>Math.abs(b-raw)<Math.abs(a-raw)?b:a);
  if(Math.abs(raw-closest)<0.04) raw=closest;
  if(speedMode==='master'){
    setSpeedMaster(raw);
    engine.setSpeedMaster(raw);
  } else {
    setSpeedPerTrack(p=>({...p,[target]:raw}));
    engine.setSpeedTrack(target,raw);
  }
}}
```
